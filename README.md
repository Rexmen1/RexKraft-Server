## ü§ù How to Contribute

Found a typo in a quest? Think a weapon is unbalanced? Want to fix a bug in the chat formatting? You can fix it yourself!

We use a standard GitHub workflow (Fork & Pull Request).

### Step 1: Fork the Repository
Click the **Fork** button in the top-right corner of this page. This creates a copy of the server files on your own GitHub account.

### Step 2: Make Your Changes
You can edit files directly in your browser or clone the repo to your PC.
1.  Navigate to the file you want to fix (e.g., `plugins/Essentials/motd.txt`).
2.  Make your edits.
3.  Commit the changes to a new branch (e.g., named `fix-typo-spawn`).

### Step 3: Submit a Pull Request (PR)
1.  Go back to the [RexKraft-Server Repository](https://github.com/Rexmen1/RexKraft-Server).
2.  Click the **Pull Requests** tab.
3.  Click **New Pull Request**.
4.  Select your fork/branch and compare it with our `main` branch.
5.  **Important:** Write a clear title and description of what you changed and why.

### Step 4: Review & Merge
Our staff team will review your request. If the changes look good and are safe for the server, we will merge them!
* *Note: Approved changes may require a server restart to take effect in-game.*
